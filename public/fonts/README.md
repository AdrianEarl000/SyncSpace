# Fonts Required

Download these free fonts from Fontshare (https://www.fontshare.com) and place .woff2 files here:

## Clash Display
URL: https://www.fontshare.com/fonts/clash-display
Files needed:
- ClashDisplay-Regular.woff2
- ClashDisplay-Medium.woff2
- ClashDisplay-Semibold.woff2
- ClashDisplay-Bold.woff2

## Satoshi
URL: https://www.fontshare.com/fonts/satoshi
Files needed:
- Satoshi-Regular.woff2
- Satoshi-Medium.woff2
- Satoshi-Bold.woff2

Both fonts are free for commercial use.
